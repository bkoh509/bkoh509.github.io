• 그래프 표현학습 (Graph Representation Learning)
    • 노드 & 그래프 분류, 링크 예측
      노드 분류
      - 그래프 구조와 노드 특성에 기반하여 개별 노드에 레이블을 할당하는 작업에 초점을 둡니다
      - 소셜 네트워크 사용자 분류, 사기 탐지, 단백질 상호작용 네트워크에서의 생물학적 기능 예측 등에 흔히 적용됩니다
      - 대표적인 기법으로는 Graph Convolutional Networks(GCNs), Graph Attention Networks(GATs), GraphSAGE 등이 있습니다

      그래프 분류
      - 그래프 전체(예: 분자 그래프가 독성이 있는지 여부)에 대한 레이블을 예측하는 것을 목표로 합니다
      - 화학, 신약 개발, 소셜 네트워크 분석 분야 등에서 유용하게 쓰입니다
      - 주로 노드 단위의 특징과 계층적 풀링 방법을 결합해 전체 그래프 표현을 학습하는 접근법이 활용됩니다

      링크 예측
      - 그래프 내에서 두 노드 간 연결의 존재 여부나 연결 확률을 예측합니다
      - 소셜 네트워크(친구 추천), 전자상거래(제품 추천), 지식 그래프(관계 발견) 등에서 광범위하게 활용됩니다
      - 노드 임베딩, 쌍대 거리 측정, 관계 기반 추론 등이 주된 기법으로 사용됩니다
    • 지식 그래프 표현/완성/검증/구축
      지식 그래프 표현
      - 개체와 관계를 벡터 임베딩으로 학습하여 의미적·구조적 정보를 포착합니다
      - TransE, DistMult, ComplEx, RotatE 등이 대표적인 기법입니다
      - 질의응답이나 추천 시스템 같은 후속 과제에도 유용하게 활용됩니다

      지식 그래프 완성
      - 지식 그래프에서 누락된 사실(엣지)을 예측하는 것을 목표로 합니다
      - 임베딩 기반 모델이나 규칙 기반 추론으로 숨겨진 관계를 찾아냅니다
      - 지식 베이스가 포괄적이고 최신 상태를 유지하도록 돕습니다

      지식 그래프 검증
      - 새로 추가된 사실이 올바르고 일관된지 확인하는 과정입니다
      - 논리적 일관성 검증과 머신 러닝 기반 이상 탐지를 결합합니다
      - 고품질·신뢰도 높은 지식 저장소를 유지하는 데 매우 중요합니다

      지식 그래프 구축
      - 비정형 또는 반정형 데이터(예: 텍스트, 테이블)로부터 지식 그래프를 생성합니다
      - 개체 인식, 관계 추출, 개체 연결, 데이터 통합 등의 단계를 포함합니다
      - 대규모 확장을 위해 NLP 파이프라인, 크라우드소싱, 자동화된 큐레이션 등이 자주 활용됩니다
  • 추천 시스템 (Recommender Systems)
    • 지식 활용 & 설명 가능한 추천
      지식 활용 추천
      - 지식 그래프나 도메인 온톨로지 같은 외부 지식을 결합하여 추천 정확도와 범위를 향상합니다
      - 아이템, 사용자, 문맥적 요인 등을 활용해 콜드 스타트 문제를 해소합니다
      - 더욱 정교한 사용자 선호 모델링과 의미적 추론을 가능케 합니다

      설명 가능한 추천
      - 특정 아이템이 추천되는 이유를 해석 가능한 형태로 제공합니다
      - 시각화 대시보드, 자연어 설명, 특성 기여도(어텐션, 특성 중요도) 등의 방식을 활용합니다
      - 추천 알고리즘의 “블랙박스” 요소를 줄여 사용자 신뢰도와 만족도를 높입니다
    • 대화형 추천, 그래프 기반 추천
      대화형 추천
      - 텍스트 채팅 또는 음성 인터페이스를 통해 대화형으로 사용자 선호도를 파악합니다
      - 실시간 사용자 피드백을 바탕으로 추천을 동적으로 조정합니다
      - 강화학습, 다중 턴 대화 모델링, 사용자 의도 파악 등의 기법을 이용합니다

      그래프 기반 추천
      - 사용자, 아이템, 보조 정보를 상호 연결된 노드로 모델링하여 풍부한 관계 신호를 포착합니다
      - 그래프 신경망이나 그래프 기반 유사도 계산 등을 활용해 추천을 생성합니다
      - 소셜 추천, 전자상거래, 지식 기반 추천처럼 복잡한 도메인에서 우수한 성능을 보입니다
  • 자연어 처리 (Natural Language Processing)
    • 질의응답, 정보검색 & 정보추출
      질의응답(QA)
      - 텍스트 단락, 지식 그래프 등 정형·비정형 데이터에서 사용자 질의에 대한 답변을 자동으로 찾습니다
      - 대규모 코퍼스를 탐색하는 오픈 도메인 QA부터 특정 도메인 지식 베이스를 활용하는 클로즈드 도메인 QA까지 포함합니다
      - BERT, GPT 같은 트랜스포머 기반 모델이나 DrQA, FiD 등 QA 특화 아키텍처가 주로 쓰입니다

      정보 검색(IR)
      - 사용자 질의에 적합한 문서나 텍스트 단락을 찾아내는 데 초점을 둡니다
      - 전통적인 TF-IDF, BM25부터 ColBERT, DPR 같은 신경 랭킹 모델까지 다양합니다
      - 검색 엔진, QA 파이프라인, 대규모 텍스트 분석 등에 핵심적으로 활용됩니다

      정보 추출(IE)
      - 비정형 텍스트에서 개체, 관계, 이벤트 등 구조화된 정보를 식별해냅니다
      - 개체명 인식(NER), 관계 추출, 이벤트 검출, 동일 지시어 해소 등을 포함합니다
      - 지식 그래프 구축, 텍스트 분석, 고급 의미 이해의 기초가 됩니다
    • 멀티모달 & 지식 강화 기반모델
      멀티모달 기반 모델
      - 텍스트, 이미지, 오디오, 비디오 등 여러 데이터 형태를 단일 표현으로 통합합니다
      - 이미지 캡셔닝, 시각 질의응답, 비디오 요약 등의 작업을 가능하게 합니다
      - CLIP, ALIGN, 다양한 비전-언어 트랜스포머 등이 대표적인 접근 방식입니다

      지식 기반 확장 모델
      - 지식 그래프나 도메인 사전 같은 외부 지식을 대형 언어 모델에 주입하여 추론력과 사실 정확도를 높입니다
      - 검증된 정보를 참조해 모델의 환각(hallucination)을 완화합니다
      - 지식 프롬프트, 개체 임베딩 활용, 지식 그래프 조회 기반 파인튜닝 등 다양한 기법을 포함합니다
    • 지식 & LLM 증류
      지식 증류
      - 대형 "교사" 모델이나 지식 원천에서 축소된 "학생" 모델로 지식을 이전합니다
      - 모델 크기와 추론 지연 시간을 줄이면서도 성능을 유지할 수 있습니다
      - 모바일 기기나 엣지 컴퓨팅 등 자원 제약 환경에서 NLP 시스템 배포 시 핵심입니다

      LLM 증류
      - GPT, BERT 같은 대형 언어 모델을 더 작고 특정 과제에 맞는 구조로 재구성합니다
      - 사전 학습의 장점을 유지하면서도 계산 자원을 크게 절감합니다
      - 레이어 스키핑, 가중치 양자화, 교사-학생 훈련 프레임워크 등이 대표적인 기법입니다
  • LLM과 그래프의 시너지 (Synergizing LLMs and Graphs)
    • 텍스트→그래프 & 그래프→텍스트 생성
      텍스트→그래프 생성
      - 텍스트 설명이나 대화를 자동으로 구조화된 그래프(예: 지식 그래프, 장면 그래프)로 변환합니다
      - 후속 그래프 분석을 위해 텍스트를 구조화된 형태로 요약하는 데 유용합니다
      - 고급 자연어 이해, 개체 연결, 관계 추출 등에 의존합니다

      그래프→텍스트 생성
      - 그래프 기반 데이터(예: 지식 그래프의 부분 구조)를 일관성 있는 텍스트나 자연어 설명으로 변환합니다
      - 그래프 데이터에 근거한 맞춤형 보고서, 챗봇, 질의응답 해설 등에 적용됩니다
      - 그래프 탐색, 콘텐츠 구성, 자연어 생성 등의 과정이 필수적입니다
    • LLM 기반 지식 활용 & 문맥 인지형 응답 생성
      지식 기반 응답 생성
      - 챗봇이나 가상 비서에서 정확하고 사실적인 응답을 위해 지식 베이스나 외부 코퍼스를 활용합니다
      - 신뢰할 수 있는 지식을 참조하여 환각(hallucination) 위험을 낮춥니다
      - 검색 결합 생성, 지식 어텐션, 메모리 증강 모델 등이 대표적 방법입니다

      문맥 인지형 응답 생성
      - 사용자 맥락, 대화 이력, 넓은 담화를 고려하여 응답을 맞춤화합니다
      - 다중 턴 대화 모델링, 사용자 선호 추적, 상황 인지 등이 포함됩니다
      - 개인화된 일관성으로 사용자 경험을 한층 향상시킵니다
    • LLM용 그래프 구조 인터랙션 (GraphRAG, 그래프 기반 LLM 에이전트)
      GraphRAG (그래프 기반 검색 증강 생성)
      - 그래프 검색 메커니즘을 대형 언어 모델과 결합해 사실적으로 정확한 콘텐츠를 생성합니다
      - 그래프 임베딩이나 구조화된 검색으로 관련 노드/엣지를 찾아 생성 조건으로 활용합니다
      - 순수 텍스트 기반 검색 대비 해석 가능하고 설명이 용이한 결과를 제공합니다

      그래프 기반 LLM 에이전트
      - 복잡한 과제에서 대형 언어 모델의 의사결정이나 행동 계획을 그래프 구조로 안내합니다
      - 단계별 추론, 다중 홉 추론, 구조화된 지식 조작을 가능하게 합니다
      - 추론 능력, 계획, 투명성을 강화하여 AI 에이전트를 혁신할 잠재력을 지닙니다
